RSA CUDA

Posted on 2022-08-24

1 文件说明

CUDA_RSA相关资料文件夹下包括：*code和参考资料*两个文件夹。

1.1 code 文件夹

该文件夹主要包括两个工程项目：libgpucrypto和*lib_gpu*。

1.2.1 libgpucrypto

该开源代码来源于KAIST开发的SSLShader。

libgpucrypto 是 SSLShader 软件的一部分，它实现了一些加密算法：AES、SHA1、使用 CUDA 的 RSA。包括几个数据结构，帮助利用 CUDA 的stream以获得更好的性能。开源项目的地址为：http://shader.kaist.edu/sslshader/libgpucrypto/

该开源代码的运行需要旧版本的cuda toolkit，不易与现在的GPU适配。以下为该代码编译运行所需的环境：

CUDA 4.0
CUDA driver : 270.41.19
CUDA toolkit : 4.0.17
CUDA SDK : 4.0.17

CUDA 3.2
CUDA driver : 260.19.26
CUDA toolkit : 3.2.16
CUDA SDK : 3.2.16

O/S
Ubuntu 10.04 LTS 64bit

CUDA stuff 下载地址：https://developer.nvidia.com/cuda-toolkit-40

openssl下载地址：https://www.openssl.org/source/

存在的问题

该代码采用了CUDA SDK中的函数进行实现，而现在的CUDA版本已经不支持该SDK，需要替换掉已经不被支持的函数部分。

该代码调用了CUDA samples里面的help_cuda.h文件，导致无法编译通过。需要对缺失函数进行重写。

openssl 版本不兼容问。在编译过程中碰到类型未定义类错误。是openssl 1.0.x版本和openssl 1.1.x版本之间不兼容导致的。需要根据最新的openssl支持的语法重写。

1.2.2 lib_gpu

该部分的代码在libgpucrypto上进行移植，可以支持在最新版本的CUDA和openssl上运行，支持：

CUDA 11

Ubuntu 20.04 LTS 64bit

Openssl 1.1.1

已解决的问题

重写CUDA SDK，替换已经不被支持的函数部分。

重写CUDA samples里面的help_cuda.h文件中被调用的相关函数。

需要根据最新的openssl支持的语法重写相关API。

支持CUDA多个stream的加密模式。

1.2 参考资料 文件夹

主要包括项目相关的学习资料，cuda c编程实战和样例代码。
